# Genius Investor
Мини-приложение Telegram, сделанное на Next.js 14 с использованием App Router.

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Запустите локально:
```bash
npm run dev
```

3. Для продакшн:
```bash
npm run build
npm start
```

Готово для деплоя на Vercel.
